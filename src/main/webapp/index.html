Welcome to the Ontology-Driven Interactive Search Environment for Earth Science (ODISEES). Users can query the ASDC semantic metadata repository for information about and access to archived data in the CERES and CALIPSO datasets. No prior knowledge of the ontology, or the data vocabulary is required.

To begin, select a category. Or, visit the help page.
